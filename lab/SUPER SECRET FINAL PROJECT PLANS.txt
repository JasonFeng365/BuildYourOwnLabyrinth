Level data grid: 5x5 array
uint_8: [7-6 unused][5-4 tile type][3-0 height]

Level code: 25 length string
Row-column order: (0, 0) (0, 1) ... (1, 0) ...
a-j: height of tile

Pixel color array: 128x128 ints = 64kb

	Draw background pixels over ball, if covering + in front of ball
